# Customer Churn Prediction and Customer Segmentation

## Project Overview

This project predicts customer churn using Machine Learning and segments customers using K-Means Clustering to support customer retention and business decision-making.

## Dataset

Telco Customer Churn Dataset

Total Customers: 7043

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* K-Means Clustering

## Machine Learning Model

Random Forest Classifier

### Performance Metrics

| Metric    | Score  |
| --------- | ------ |
| Accuracy  | 78.35% |
| Precision | 57.39% |
| Recall    | 71.66% |
| F1 Score  | 63.73% |
| AUC Score | 85.11% |

## Customer Segmentation

K-Means clustering was used to segment customers into four groups:

1. High-Spending New Customers
2. Premium Loyal Customers
3. Budget Loyal Customers
4. New Low-Value Customers

## Business Insights

* Customers with shorter tenure are more likely to churn.
* High monthly charges increase churn probability.
* Contract type strongly affects customer retention.
* Premium loyal customers generate the highest revenue.
* Customer segmentation enables targeted retention strategies.
